Stock Sentiment Analysis
Ce projet scrute des données provenant de sites populaires comme Finviz, Reddit et Twitter afin de prédire la direction des actions en utilisant l'analyse des sentiments avec le modèle FinBERT. L'objectif est d'analyser les sentiments exprimés sur ces plateformes pour déterminer si les actions vont probablement monter ou descendre.

En plus, une application web construite avec Next.js permet à l'utilisateur d'entrer un ticker d'action et le nombre de jours pour récupérer les données. L'interface affiche les résultats de la prédiction, ainsi que des visualisations graphiques des sentiments analysés.
